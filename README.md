## Source for tinocardenes.com

I made this website using Next.js as a proof that coding from scratch is about as efficient and customizable as Wix. 
The underlying motivation is to save money on serving a static site, while learning a modern web framework. 

## Deploying
If for whatever reason you want to run this locally instead of visiting <a href="https://www.tinocardenes.com">tinocardenes.com</a>, then use `pnpm run dev` to run on your localhost. 

At time of writing, I currently deploy using Vercel connected to my GoDaddy domain. The main benefit being that it auto-deploys on git push, and for minor traffic is a free static option. They also handle all DNS linking.
